## Planetary Exploration Rover using A* search
This project is toy Planetary Exploration Rover which is implemented using the A* search to explore different paths and find out the optimal path. The environment is made up of different types of terrains such as rocky, sandy, flat, cliff and hazardous terrains like sandtrap and radiation. There are also charging stations for the rover and start state and the goal state. Every exploration move has a cost which varies according to the terrain type. We implemented the A* search using 4 different types of heuristics - Weighted Manhattan, Euclidean, Terrain-cost Weighted and Battery aware heuristics and the performance and the efficiency is calculated on the basis of explored nodes, total path cost and computation time. A battery aware model is implemented so that it can monitor the energy consumption dynamically and detour to the charging station when battery falls below a certain threshold. After recharging it again resumes its A* search for the goal state. If the battery is not sufficient then we also implemented the self charging system which charges a certain amount of battery which is enough to move to the next exploration state. Lastly, we have tried to implement it visually so that we can clearly state how the agent is performing and in the end of the visualization we have provided the optimal path.
